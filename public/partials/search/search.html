Tips
You can bookmark this page :
  • http://{{ hostname }}:{{ port }}/search/<environment>/<soapAction>/<fromDate>/<toDate>/
  • Default page : /search/all/all/all/today/ : all requests
  • Change start date : /search/all/all/2012-12-10/today/: all requests from 2012-12-10
  • Advanced : /search/dev/login/yesterday/today/ : Results for environment dev, soapAction login from yesterday, to today
You can use advanced search :
  • Search at least two words, word1 or word2 : "word1|word2"
  • Search a word with unknown character : "wo.d1"
